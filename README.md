# Bruce JS Designer

**Bruce JS Designer** is an HTML-based tool that makes designing JavaScript apps easier. It provides a live canvas for creating shapes, text, and graphics, with full control over properties like position, size, color, and device display simulation. Perfect for prototyping and generating display code for JS devices.

## Features

- Add and edit shapes: rectangles, lines, pixels, circles, text
- Set device display (e.g., M5StickC, LilyGO T-Embed)
- Live preview of your design on a canvas
- Responsive UI for both PC and mobile
- Color picker, sliders, and interactive controls
- Credits link to GitHub profile

## Usage

1. Open [bruceJSdesigner](https://tebin623.github.io/Bruce-JS-Designer/) in a web browser.
2. Select the device you want from the device picker below the canvas.
3. Use the buttons to add shapes to the canvas.
4. Adjust properties using sliders, color picker, and input fields.
5. Click Export JS to download your generated code as a ".js" file.
